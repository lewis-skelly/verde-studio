# Verde Studio — Fotografie-Portfolio

Demo-Website (Referenzprojekt fürs Portfolio).
Reines HTML/CSS/JS — einfach `index.html` im Browser öffnen.

## Anpassen
Alles steht in der `index.html` (Inhalt + Design + Skript in einer Datei).
Farben oben im `:root`-Block, Texte direkt im HTML.

## Online stellen
Kostenlos via Netlify Drop, Vercel oder GitHub Pages.
